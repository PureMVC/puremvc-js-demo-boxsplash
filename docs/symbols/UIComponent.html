Class UIComponent

An ultra-light framework class used as a wrapper and psudo-extenstion of the MooTools Native Element implementation. Its primary goal is to encuorage the addition of elements to the DOM in a top-down fashion.
Defined in: boxsplash.js.

Class Detail
UIComponent(element, properties)
Parameters:
{mixed} element
String or Element to become the subject of ths instance.

If a string is passed, an element ID is assumed first. if the element ID is not found in the DOM, a tag name is assumed and MooTools will attempt to create that Element.

If an Element is passed in, it is left as-is.

{Object} properties
The Object containing the default properties to be set on the Element.

Method Detail
{UIComponent} addChild(child)
Wrapper for the MooTools Element.grab() that serves 2 primary purposes.
  1. Appends the child UIComponent's DOMElement to this one.
  2. calls the child's initialization routines to gaurantee all elements are added to the DOM in a 'top down' fashion. (else IE will leak like a siv)
Parameters:
{UIComponent} child
The child to append.
Returns:
{UIComponent} this object - can be used for chaining
